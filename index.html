About

My passion for anything IT started over 10 years ago when Windows XP still rulled the desktop computing realm.

I have since taken on multiple computing self-paced training courses ranging from windows active directory administration (MSCE) to different database technologies like Oracle, network management using Cisco (CCNA) and finally linux system administration. As the industry evolved, I then got involved in web developement on a freelance basis.

My taking on a 'MSC Computer Science' is in an effort to consolidate my knowledge while learning more practicle and applicable skills in the industry.

Work Experience in a nutshell

  • My current job as a 'Remote IT Operations technichian' with a US based company called Vidvita LLC sees me work with various platforms and IT systems to ensure smooth day to day operations.
  • Previous to my current role, I had worked in various capacities gathering a range of skills along the way. Some of the roles I have been in are highlighted below;
    • Project consultant at Tanzania's governemnt data centerNIDC
    • Freelance web developer building web platforms for SMEs in PHP using the Laravel framework
    • Network Engineer in a Tanzanian media monitoring company called Push Observer
    • System Adminstrator in a startup company dealing in telecom VAS products
    • Contact Center Executive in Orange Telecom Uganda

Launching into Computer Science

This is the first module in the master's program. It is meant as a high level overview or an 'ice breaker' for some of the topics to be covered in greater detail during the program.

learning objectives of this module

  • Identify and explain the architecture, structure and functionality of basic components of computer system
  • Demonstrate a critical understanding of core data structures and programming concepts, including algorithm computability
  • Critically evaluate the functionality of different types of software, i.e., operating system, utility programs, languages and applications
  • Critically appraise the emerging trends in the field, such as cloud computing, big data, cyber security, and the professional and ethical requirements for dealing with such contemporary computer-based technologies.
